# Step-by-step installation instructions

## Installation
**a.** Install [pytorch](https://pytorch.org/)(v1.8.1+cu111).

**b.** Install mmcv-full==1.6.0  mmdet==2.24.0.

**c.** Install outher requirements.
```shell
pip install -r requirements.txt
```
**d.** Install Pro3D (gpu required).
```shell
python setup.py develop
```
